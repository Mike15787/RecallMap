# P0-6｜前端 — 學習地圖視覺化

# 功能描述

呼叫 `GET /v1/sessions/{id}/map` 取得盲點分析結果，以列表方式顯示三個區（已知 / 模糊 / 盲點），讓使用者點擊盲點進入蘇格拉底對話。

---

# 輸入 / 輸出

|  | 說明 |
| --- | --- |
| **API 輸入** | `GET /v1/sessions/{id}/map` |
| **畫面輸出** | 三個分區的概念列表，每個盲點可點擊 |
| **使用者動作** | 點擊盲點 → 進入對話頁面 |

---

# 實作範圍

## ✅ 包含

- 進入頁面後自動呼叫 map API（顯示 loading）
- 三區分組顯示（盲點區 / 模糊區 / 已知區）
- 每個概念顯示：名稱、信心度（百分比）、重複次數
- 盲點區的概念有「開始理解」按鈕 → 進入對話頁
- 「重新分析」按鈕 → 呼叫 `?force=true`
- 分析中的 loading 提示文字
- 地圖為空時的空狀態畫面（提示去上傳材料）

## ❌ 不包含

- 複雜的圖形視覺化（節點圖、泡泡圖）
- 拖曳移動節點
- 地圖歷史版本比較
- 匯出地圖功能

---

# UI 文字規範

| 情境 | 文字 |
| --- | --- |
| 分析中 | 「AI 正在分析你的學習盲點...」 |
| 盲點區標題 | 「還不太懂的概念」 |
| 模糊區標題 | 「有點印象，但不夠清楚」 |
| 已知區標題 | 「已經理解的概念」 |
| 空狀態 | 「還沒有上傳學習材料，去上傳看看？」 |

---

# 信心度顯示方式

```
confidence → 顯示
0.0–0.44 → 紅色標籤「需要加強」
0.45–0.74 → 黃色標籤「有點懂」
0.75–1.0 → 綠色標籤「理解中」
```
